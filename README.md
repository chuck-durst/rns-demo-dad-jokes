# React-next-starter

Ce starter prend en charge les fonctionnalités suivantes:

- routing SSR
- server node (expressJs)
- gestion du cache
- redux
- custom loading
- sass
- async module & component
- custom webpack config
- fake api
- i18n
- tests & CI/CD

## Requirements

- nodeJs ^9.8.0


## Installation

### Installer les dépendances du projet
`npm i`

### Installer nextJs en global

`npm i -g next`
